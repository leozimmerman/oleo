code

open-source tools for audiovisual creation

sonoscopio

audio analysis with an integrated timeline that sends data over network using OSC and renders information in .xml files.

essentia plugin

VST plugin for analyizing audio descriptors in real time and send them over network using OSC.

essentia light

VST plugin for analyizing RMS, Loudness and Power and send data over network using OSC

midi sender

VST plugin for sending multi channel midi events over OSC.

parameter sender

VST plugin for sending parameters values over OSC. Values can be automated using any host DAW.

audioAnalyzerCHOP

TouchDesigner's CHOP for analyzing audio descriptors.
